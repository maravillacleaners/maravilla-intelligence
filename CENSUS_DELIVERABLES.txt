STEP 3: RESTART SERVER (1 minute)
─────────────────────────────────────────────────────────────────────────────

================================================================================
                         API ENDPOINTS
================================================================================

GET /api/scrapers/census
────────────────────────────────────────────────────────────────────────────
Query Parameters:
  state (required)  - State code (FL, CA, NY, TX, etc.)
  county (optional) - County name
  zip (optional)    - ZIP code
  city (optional)   - City name
  naics (optional)  - NAICS code for industry

Response:
  {
    "success": true,
    "demographics": { ... },
    "businessContext": { ... },
    "source": "census_api|cached|fallback",
    "cacheHit": true|false,
    "timestamp": "ISO 8601"
  }

Example:
  curl "http://localhost:3000/api/scrapers/census?state=FL&county=Lee&zip=33901"

POST /api/scrapers/census
────────────────────────────────────────────────────────────────────────────
Request Body:
  {
    "state": "FL",
    "county": "Lee",
    "zip": "33901",
    "naics": "561720"
  }

Response: Same as GET

GET /api/scrapers/census?action=metrics
────────────────────────────────────────────────────────────────────────────
Returns:
  {
    "success": true,
    "cache": {
      "size": 5,
      "entries": [...]
    }
  }

GET /api/scrapers/census?action=clear-cache
────────────────────────────────────────────────────────────────────────────
Returns:
  {
    "success": true,
    "message": "Cache cleared",
    "timestamp": "ISO 8601"
  }
