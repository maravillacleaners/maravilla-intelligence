KEY FEATURES IMPLEMENTED

Location Filtering
   - Florida only (11 counties supported)
   - Active opportunities only

NAICS Code Filtering
   - 561720 (Janitorial Services) - primary target
   - 561710, 561730, 561790, 561110, 561210 - related codes
   - Weighted scoring by relevance

Intelligent Scoring (1-100)
   - NAICS match (0-50 points)
   - Keyword matching (0-50 points)
   - Set-aside bonus (+15 points)
   - Deadline urgency (0-20 points)
   - Auto-exclusion (hazmat, asbestos, etc.)

Signal Strength Classification
   - HIGH (70+) - High confidence
   - MEDIUM (50-69) - Moderate match
   - LOW (<50) - Weak signal

Contact Information Extraction
   - Primary POC name, email, phone
   - Fallback to secondary contact
   - 80%+ capture rate expected

Airtable Integration
   - 17-field schema mapping
   - Direct writes to Opportunities table
   - Duplicate prevention (notice ID based)
   - Error recovery with partial results

Production-Grade Features
   - Rate limiting (300ms SAM + 100ms Airtable)
   - Error handling with detailed logging
   - Configurable lookback period (1-90 days)
   - Timeout protection
   - Graceful degradation on failure
