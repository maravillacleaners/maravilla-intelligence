================================================================================
  ENDPOINTS READY FOR USE
================================================================================

POST /api/scrapers/sunbiz
   Input:  {company_name: string, county?: string}
   Output: {companies[], saved_to_airtable[], count, source, timestamp}

POST /api/scrapers/property-records
   Input:  {address?, owner_name?, county?, avatar_id?}
   Output: {properties[], saved_to_airtable[], count, source, timestamp}

POST /api/scrapers/contact-finder
   Input:  {company_name?, company_domain?, email?, company_id?}
   Output: {contacts[], saved_to_airtable[], count, sources[]}

POST /api/scrapers/investigator-merge ⭐ RECOMMENDED
   Input:  {name: string, entity_type?: string, county?: string}
   Output: {
     avatar_profile: {
       name, entity_type, confidence_score, investigation_score,
       business_registrations[], properties[], contacts[],
       federal_contracts[], relationships[], risk_flags[], etc.
     },
     saved_to_airtable: string,
     investigation_complete: boolean
   }

================================================================================
  KEY FEATURES
================================================================================

✅ Multi-source data aggregation
   - Combines data from 4+ different scrapers
   - Deduplicates automatically
   - High-confidence scoring

✅ Automatic risk flagging
   - Detects shell companies
   - High property portfolios
   - Federal contract exposure
   - Missing data patterns

✅ Investigation scoring
   - 0-100 score based on data completeness
   - Formula: confidence + properties*5 + contacts*3 + federal*20 + sources*2
   - Helps prioritize leads

✅ Relationship mapping
   - Discovers connections between entities
   - Competitor detection
   - Partnership potential

✅ Rate limit compliance
   - Respects API rate limits
   - Exponential backoff on errors
   - Batch operations support

✅ Graceful degradation
   - Works with public APIs if no keys configured
   - Fallback sources if primary fails
   - Partial results returned if some sources down
